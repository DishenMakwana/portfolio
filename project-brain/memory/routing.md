# Routing Map - routing.md

## App Routes Layout
*   `/` - Main Dashboard Overview
*   `/allocation` - Portfolio Asset & Cap Allocation
*   `/family` - Per-member CAGR/XIRR breakdown
*   `/holdings` - Comprehensive portfolio positions
*   `/sips` - Active/Paused Systematic Investment Plans
*   `/uploads` - Sheet Upload Calendar & Snapshot Registry
*   `/mapping` - Missing API Scheme Mapping dashboard
*   `/fund/[id]` - Fund fact-sheet, transaction history, volatility metrics (Beta, Sharpe, StdDev), and NAV charting
