# Security Review - security-review.md

## Checklist
*   [ ] SQL Injection: Are all queries constructed using parameter bindings (Drizzle standard)?
*   [ ] XSS: Are string values escaped or safely formatted during presentation?
*   [ ] Authentication: Are data requests scope-locked to context parameters?
*   [ ] Secret Exposure: Are passwords, raw keys, or sensitive PAN strings excluded from logs?
